<!DOCTYPE html> - це декларація типу документа, і вона не може бути замінена. Вона завжди використовується для вказання версії HTML, і в даному випадку це HTML5.
<html lang="uk"> - Тег <html> визначає початок та кінець HTML документа. Атрибут lang вказує мову документа. Його можна замінити на будь-яку іншу мову, наприклад, lang="en" для англійської мови.
<meta charset="UTF-8"> - Вказує кодування документа. Можна використовувати інші кодування, але UTF-8 загалом є стандартом для веб-документів.
<meta name="viewport" content="width=device-width, initial-scale=1.0"> - Встановлює налаштування відображення на мобільних пристроях. Його можна змінити в залежності від потреб вашого дизайну.
<title>Контакти та підтримка</title> - Заголовок веб-сторінки. Можна змінити текст заголовка на будь-який інший.
<link rel="stylesheet" href="style.css"> - Посилання на зовнішній CSS файл для стилізації сторінки. Його можна замінити на тег <style>, щоб вбудувати стилі безпосередньо в HTML файл.

.rounded li{
    position: relative; /* Встановлюємо позицію відносно */
    display: block; /* Блочний елемент */
    font: 30px "Trebuchet MS", "Lucida Sans"; /* Розмір та шрифт тексту */
    padding: .4em .4em .4em 2em; /* Відступи всередині li */
    margin: .5em 40px; /* Зовнішні відступи */
    background: #DAD2CA; /* Колір фону li */
    color: #444; /* Колір тексту */
    text-decoration: none; /* Видалення підкреслення */
    border-radius: .3em; /* Закругленість кутів */
    transition: .3s ease-out; /* Плавна зміна стилів при наведенні */
}

 .rounded li:hover{
    background: #E9E4E0; /* Колір фону при наведенні 
}

.rounded li:hover::before{
    transform: rotate(360deg); /* Обертання при наведенні 
}
